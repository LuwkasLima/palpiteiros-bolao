---
name: bugfix-workflow
description: Investigate and fix bugs using reproduction evidence, root-cause analysis, and verification.
---

Use this skill when:

- The task is a bug report
- There are failing tests, logs, or runtime errors
- The root cause is not yet known

Workflow:

1. Gather evidence: logs, stack traces, failing tests, reproduction steps.
2. Form a root-cause hypothesis.
3. Confirm the hypothesis before making broad changes.
4. Implement the smallest correct fix.
5. Run targeted verification.
6. Record the cause, fix, and verification outcome in the task notes.
